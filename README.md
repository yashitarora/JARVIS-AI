## 📋 Overview

A personal AI assistant inspired by Iron Man's JARVIS, featuring wake word detection, voice commands, and natural language processing for task automation.

---

## ✨ Features

- ✅ **Wake Word Detection**: Activates on "Hey JARVIS"
- ✅ **Voice Commands**: Process and execute spoken commands
- ✅ **Natural Language Understanding**: Interprets user intent
- ✅ **Interactive HUD**: Iron Man-style interface
- ✅ **Task Automation**: Web search, reminders, calculations

---

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/yashitarora/JARVIS-AI.git

# Navigate to directory
cd JARVIS-AI

# Install dependencies
pip install -r requirements.txt

# Run JARVIS
python jarvis.py
```

---

## 🎯 Commands

```
"Hey JARVIS" - Activate
"What time is it?" - Get current time
"Search for [query]" - Web search
"Calculate [expression]" - Math operations
"Open [application]" - Launch apps
```
